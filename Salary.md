#### 2. 월급 계산
- 입대일 포함된 경우
   - {(기본급) X (입대일 ~ 30,31일 / 달 전체일수)} + (기타 추가금)
   
- 전역일 포함된 경우
   - {(기본급) X (1일 ~ 전역일 / 달 전체일수)} + {(교통비) X (이전달 출근한 일수)}</br> + {(식비) X (이전달 출근한 일수)} + (기타 추가금)
   
- 입대일, 전역일 포함되어 있지 않을 경우
   - (기본급) + {(교통비) X (이전달 출근한 일수)} + {(식비) X (이전달 출근한 일수)}</br> + (기타 추가금)
